Below are the config values and how to use them for Confirm Thread Creation

### Configuration Keys:

**RED** - confirm\_thread\_creation\_title **YELLOW** - confirm\_thread\_response **GREEN** - confirm\_thread\_creation\_accept **BLUE** - confirm\_thread\_creation\_deny

![](\images\modmail-setup\basic-config\files%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FYrRMaDL0XmNCuOANKDdD%252Fuploads%252FusjwkgE4u5utjte9zA73%252Fticket%2520confirm.png)

### Enable Confirm Thread Creation

To enable Confirm Thread Creation use **\[p]config set confirm\_thread\_creation true**.

![](https://andreass-organization-3.gitbook.io/~gitbook/image?url=https%3A%2F%2Fmedia.discordapp.net%2Fattachments%2F993182447476953200%2F1333158639887978588%2Fimage.png%3Fex%3D6797e03c%26is%3D67968ebc%26hm%3Dc2e891f694bdb45b8287767ad5fc955ae5e9cc71527ca02ddc85d76e900c9ffc%26%3D%26format%3Dwebp%26quality%3Dlossless%26width%3D588%26height%3D285\&width=768\&dpr=4\&quality=100\&sign=d60de8db\&sv=2)

### Title

RED - **\[p]config set thread\_creation\_title Confirm Ticket Creation** In the title you can also include an emoji if you wish

### Response

YELLOW - **\[p]config set confirm\_thread\_response Please confirm ticket creation. Click the button accordingly** In the response, aka the embed description, you can add quick info for customers before support staff claim the ticket. Please be reminded they are limited to **4096 characters**.

### Accept & Deny Buttons

GREEN **\[p]config set confirm\_thread\_creation\_accept** ***mentionyouremojihere***

BLUE **\[p]config set confirm\_thread\_creation\_deny** ***mentionyouremojihere***
