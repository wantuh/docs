Examples can be found below on how to use config and the value keys.

### To see all the config options available

Run **\[p]config options** you will be able to use the buttons and scroll through all the options available.

### Use config

To use any config key, you will need the exact config\_key. It will always be **\[p]config set config\_key new\_value**. **EXAMPLE**: To update your anon\_username. **\[p]config set anon\_username Wantuh's AI**.
